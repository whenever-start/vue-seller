# vue-seller

## 做什么？

模仿饿了么卖家模块。

## 技术栈

- 样式：`less`
- 路由：`vue-router`
- 数据来源：`mock`
- 数据处理：`全局变量`
- ajax：`axios`
- 屏幕自适应：`vw + rem`
